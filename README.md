# La vie des plantes

Site e-commerce réalisé avec **WordPress** dans le cadre d’une formation.  
Le site présente une boutique de kits de plantes biologiques destinés a une population jeune entre 25 et 35 ans, plutôt urbaine..

---

## Technologies utilisées

- **WordPress**
- **Elementor** (mise en page)
- **WooCommerce** (boutique en ligne)
- **WPForms** (formulaire de contact)
- **ElementsKit** (header / footer)

---

## Pages du site

- Accueil  
- Boutique  
- Page produit  
- À propos  
- Contact (avec formulaire et Google Map)  
- Mentions légales  
- Politique de confidentialité  
- Politique de remboursement et retours  

---

## Fonctionnalités

- Affichage des produits avec images, descriptions et prix
- Ajout au panier via WooCommerce
- Pages panier et commande fonctionnelles
- Formulaire de contact opérationnel
- Google Map intégrée sur la page Contact
- Footer personnalisé présent sur toutes les pages
- Site entièrement responsive (desktop, tablette, mobile)

---

## Design & mise en page

- Mise en page réalisée avec **Elementor**
- Palette de couleurs cohérente avec les consignes :
    - #F97068

---

## Installation (en local)

1. Installer un environnement local.
2. Installer WordPress
3. Importer la base de données si nécessaire
4. Placer les fichiers du projet dans le dossier WordPress
5. Activer le thème et les extensions utilisées
6. Adresse pour visiter le site : ** http://localhost/la_vie_des_plantes/ **

---

## Remarques

- Le site est fonctionnel en local
- Les moyens de paiement ne sont pas activés (non requis pour le projet)
- Projet réalisé dans le respect des consignes demandées
